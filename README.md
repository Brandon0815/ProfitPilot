# ProfitPilot
Tracks costs/profits and also gives tips using Gen AI 

## Features
- Real-time profit dashboard
- CSV data import
- Interactive charts
- AI-powered insights


## Tech Stack
- HTML5, CSS3, JavaScript, Python
- pandas for CSV processing
